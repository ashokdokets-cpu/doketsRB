# ResumeAI Pro — AI Resume Builder with Job Targeting

Build ATS-optimized resumes with AI-powered job targeting. Match your skills to any job description, auto-generate tailored content, and export job-ready PDFs.

## Features

- 🎯 **AI Job Targeting** — Paste any job description and get a match score, missing skills, and auto-generated content
- 📝 **Resume Builder** — Live editor with real-time preview
- 📊 **ATS Scoring** — 5-point analysis with composite scoring
- 📄 **PDF Export** — One-click professional PDF generation
- 🎨 **8 Templates** — Modern, Classic, Creative, Executive, and more
- 💾 **Auto-Save** — LocalStorage persistence
- 📱 **Responsive** — Works on all devices

## Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- Tailwind CSS (CDN)
- jsPDF + html2canvas (PDF generation)
- Lucide Icons
- Single Page Application (SPA) architecture

## Deployment

This is a static site. Deploy to any static hosting:

### Vercel (Recommended)
1. Push this repo to GitHub
2. Import to Vercel
3. Add custom domain: `doketsrb.com`
4. GoDaddy DNS: A record → `76.76.21.21`

### Netlify
Drag and drop the folder to netlify.com
